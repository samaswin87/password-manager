# Password Manager

Password Manager to maintain the user passwords

### Installed features
- Devise (with invitable module)
- CanCanCan
- Inherited Resources
- Paperclip
- Data Migrator
- Simple Form
- Slim
- AdminLTE
- Bootstrap
- FontAwesome
- Simple Navigation
- Toastr
- Notify.js
- jQuery Mask
- SweetAlert
- LazyLoad (pictures)
